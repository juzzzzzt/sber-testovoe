# Задание 2. Агент для генерации клиентской презентации


Спроектируйте и реализуйте агента, который по запросу пользователя генерирует презентацию о клиенте. Пример запроса: «Подготовь презентацию о РЖД».

На входе есть папка `data/` с данными по клиентам, финансовым продуктам, внутренним заметкам и новостям. Нужно использовать эти данные, чтобы сгенерировать презентацию о выбранном клиенте.


Ключевые требования:
- результат должен быть именно `.pptx`, а не только текстовый план;
- презентация должна быть аккуратно оформлена: единый стиль, нормальные заголовки, читаемый текст, и т.д.;
- Стек: RAG + OpenSearch + GigaChat API;
- в презентации должны быть подобраны финансовые продукты, которые подходят именно этому клиенту;
- в презентации должны быть новости, относящиеся именно к этому клиенту.

Формат сдачи: код прототипа, инструкция по запуску и 1–2 примера сгенерированных презентаций по клиентам из базы.

В качестве ответа нужно прислать ссылку на Git-репозиторий с решением.
