# Задание 1 Цитируемость новостей

## Контекст

У Вас есть датасет новостей из различных информационных источников за день. По каждой новости известно:

- заголовок 
- текст новости
- источник


## Задача

Нужно для каждой новости (или для группы новостей про одно событие) определить её цитируемость - насколько эту новость растиражировали источники.


Ваша задача - придумать и написать код, используя `data/news.csv`.

В качестве ответа нужно прислать ссылку на Git-репозиторий с решением.
