#  Анализ цитируемости новостей (Task 1)

## Задача
Для каждой новости (или группы новостей об одном событии) определить цитируемость — насколько новость растиражировали разные источники.

## Методология
1. **Предобработка**: объединение заголовка и первых 200 символов текста, lowercasing, удаление спецсимволов.
2. **Векторизация**: `TfidfVectorizer` (max_features=2000, ngram_range=(1,2)).
3. **Группировка**: косинусное сходство → матрица расстояний → `DBSCAN(eps=0.25, min_samples=2, metric='precomputed')`.
4. **Метрика цитируемости**: `citability_score = news_count × unique_sources` внутри каждого кластера.

## **Результаты**
- Всего новостей: 1880
- Выделено событий (кластеров): 77
- Формат вывода: `citability_results.csv` (кластер, количество публикаций, уникальные источники, score, пример заголовка)
